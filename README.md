# SPA Starter

The bare minimum deployable app. Your first step to building with Gista.js.

## Quick Start

### 1. Deploy to Vercel

Click "Use this template" on GitHub, then import your new repo to Vercel.

### 2. Run locally

```bash
pnpm install
pnpm dev
```

### 3. Vibe and ship

1. Open `app/routes/index.tsx`
2. Delete the green "You're Live!" section
3. Tell your AI what to build — or just start editing
4. Watch Vite hot-reload as you go
5. Commit and push — Vercel auto-deploys

## What's included

- React Router v7 (SPA mode)
- Tailwind CSS v4 + daisyUI v5
- Vite dev server (hot reload)
- Vercel-ready deployment

## What's NOT included

- No database
- No authentication
- No environment variables
- No external services
